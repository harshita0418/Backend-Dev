# Backend-Dev

This repository contains backend development topics and assignments organized in separate branches.

## Repository Structure

Each topic has its own dedicated branch. **Do not merge any branch into main.**

### Available Branches:

1. **unix-commands-git-github** - Unix commands and Git/GitHub fundamentals
2. **nodejs-programming** - Node.js programming concepts
3. **file-system-operations** - File system operations in Node.js
4. **express-framework** - Express.js framework basics
5. **restful-apis-routing** - RESTful APIs and routing
6. **introduction-to-databases** - Database fundamentals
7. **middleware-express-session-cookies** - Middleware, sessions, and cookies
8. **authentication-authorization** - Auth implementation
9. **rest-security** - REST API security
10. **deployment** - Application deployment

## Instructions

- All branches are created from the main branch
- Each branch contains topic-specific content
- Work only in the respective topic branch
- Do not merge branches into main

## How to Switch Branches
```bash
git checkout <branch-name>
```

Example:
```bash
git checkout nodejs-programming
```
